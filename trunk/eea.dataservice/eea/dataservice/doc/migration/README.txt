Dataservice Plone CMS product
-----------------------------

In this directory you will find available documentation describing the existing Dataservice system and documentation regarding the migration into the new Plone-based system.


Files descriptions:

1. dataservice-documentation.rtf
contains the intial documentation describing the dataservice CMS and metadata structure. There are also comment about plone migration considerations. 

2. dataservicemodel.zargo
ArgoUML (argouml.tigris.org) diagram for Dataservice implementation in Plone. It is intended to be used with ArchGenXML (plone.org/products/archgenxml) to  generate the product initial code automatically.
