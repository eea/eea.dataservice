Organisations control panel
=====================================================

This package consist in a portlet displaying a list of available reports
of the organisations:

 * Quick overview
     - uses a basic link checker to display external URL status
     - possibility that references can be transferred from one organisation
         to another
 * Duplicate organisations
 * Organisations with wrong title
     - listing organisations with either empty title or containing an URL
         instead of a title
 * Organisations with no reference
